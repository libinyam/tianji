01 / 创意名称与介绍

一颗引路星，照亮从理论到作品的路径

天玑 TIANJI

天玑是一个面向大学生与跨专业学习者的 AI 学习与项目共创社区，重点帮助数学、物理、计算机、金融等专业中想转向 AI、金融科技或应用开发的学生，从"只会学理论"逐步走向"能做项目、会协作、能产出"。

我想解决的问题是：很多学生想进入 AI 方向，但缺少系统指导。无论是数学、物理等理科专业，还是金融、工科等应用专业，学校课程普遍偏理论，真正涉及 GitHub、PowerShell、Codex、Claude Code、项目部署、论文落地和产品开发的内容很少。许多同学知道 AI 很重要，却不知道第一步该学什么、工具怎么配、项目怎么做。

我想到做这个，是因为我自己也经历过这种阶段。一开始我连 GUI、PowerShell、GitHub 都不熟悉，Codex 和 Claude Code 也是靠看大量视频、反复做小项目才慢慢学会。我发现，对很多学生来说，最缺的不是学习能力，而是清晰路线、真实案例和有人带着做项目的环境。

天玑的产品形态是一个论坛型 Web 社区，包含资源库、讨论区、灵感广场、协作工坊等模块，用来整合 AI 学习资料、工具配置教程、项目案例和协作任务，让学生可以从简单项目开始，逐步建立自己的技术能力和作品集。

典型使用场景

想学机器学习但不知道资料顺序 想用 GitHub 但不会上传项目 想用 Codex / Claude Code / Trae 但不会配置 想把专业知识做成项目但不会拆解任务 想部署网页却看不懂报错 想找搭档协作开发

当前三大痛点

01

资料太分散

网上教程、视频、文档很多，但质量参差不齐，新手很难判断哪些该先学、哪些工具真正重要。

02

学校教学和真实项目之间有断层

很多学生学过数学、编程或算法，但不会把知识做成可展示的作品，也不知道如何从一个想法走到项目 Demo。

03

缺少同伴和指导

AI 工具发展很快，但很多学生甚至不知道 GitHub、Codex、Claude Code、MCP、部署平台等工具的存在，更不知道如何把它们组合起来完成学习和创作。

03 / 价值与意义

让每一份专业积累，都变成真实可用的作品

效率提升

降低新手入门成本

把分散的 AI 学习路径、工具教程、项目案例和讨论经验集中到一个社区，用户不需在大量视频和文章中反复试错，而是根据自己阶段找到对应资源和可执行任务。

社会价值

赋能非计算机专业学生

帮助非计算机专业学生获得 AI 时代所需的实践能力。无论是数学、物理等理科，还是金融、工科等应用学科，传统成长路径都高度依赖天赋、导师和资源，而 AI 正在改变学习与科研方式——学生需要一条新路径：把本专业的理论积累，借助 AI 工具转化为可完成的项目、可写的文档、可协作的开发，最终变成真实作品。

长期价值

"从入门到产出"的学习共同体

学生可以在这里学习工具、提出问题、分享项目、寻找合作者，也可以通过一个个小项目慢慢建立自己的技术路线和信心，让理论能力转化成真实作品。
